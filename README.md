# Getting Started
This is a demo application for a hiring process

### About the project
This application is built using: 
- Java 8
- Spring Framework
- JPA for data persistence
- H2 as an in-memory database

### Requirements
- Java SDK 8
- Maven

### Running the project
To run the application:
``` 
mvn clean spring-boot:run
```
To run the tests:
``` 
mvn test
```
